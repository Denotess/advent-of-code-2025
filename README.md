# Advent of Code 2025 Solutions in Go

## Prerequisites
- Go 1.20 or newer installed
- Clone this repository

## Usage
1. Clone the repository:
	```
	git clone https://github.com/Denotess/advent-of-code-2025
	```
2. Place your puzzle input in the appropriate `input.txt` file for each day (e.g., `day1/input.txt`).
3. Run the solution for a day:
	```
	cd day1
	go run main.go
	```
4. The answer will be printed to the terminal.
